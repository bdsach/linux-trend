NuTyX


Last Update: 2026-04-10 12:50 UTC

  • OS Type: Linux
  • Based on: LFS
  • Origin: Switzerland
  • Architecture: x86_64
  • Desktop: GNOME, KDE Plasma, Xfce
  • Category: Desktop, Live Medium
  • Status: Active
  • Popularity: 163 (98 hits per day)
NuTyX is a French Linux distribution (with multi-language support) built from Linux From Scratch and Beyond Linux From Scratch, with a custom package manager called "cards". The package manager can install individual binary packages, a group of related binary packages (e.g. desktop packages, such as KDE or Xfce), and compile source packages from "ports". The distribution is designed for intermediate and advanced Linux users.

 Distro seems good on reading and it is very easy to install.

Just make sure you create your partitions first and format them.

Unfortunately thi is another distro that fails on the amd devices.

Install went smoothly and was exceptionally quick.

It uses an old debian style installer but as soon as the os booted, thats where the problems began.

The mouse drivers did not install.

I chose the xfce version and unfortunately there is no network manager provided in the distro either so I could not navigate up to the wifi to connect it.

Its a thin book so does not actually have a lan card.

Fairly basic thing to expect, a mouse driver on a 6.14 kernel, its kinda expected.

Instead of navigating iw commands, i chose a different distro to work on instead because i knew that trying to get a mouse driver was going to be another hurdle i just dont have time to work at.

I did not test on intel chipset, im sure it will work fine but stay clear if you have amd.

 NuTyX is an incredibly fast distribution. Installation is very easy despite the old 90s style installer interface. NuTyx is very light and is built from scratch by the developers. Installing basic packages (browser, htop etc.) is very easy. The cards repository is constantly being developed and new packages are added from time to time, which can be installed with a simple sudo cards install command. I personally highly recommend this distribution for browsing the internet or watching movies or listening to music, because if you want to install something more than is in the cards repository you have to be an advanced user. I recommend this distribution.
