CentOS


Last Update: 2026-04-07 10:11 UTC

  • OS Type: Linux
  • Based on: Fedora, Red Hat
  • Origin: USA
  • Architecture: aarch64, ppc64le, x86_64, x86-64-v2
  • Desktop: GNOME
  • Category: Desktop, Live Medium, Server
  • Status: Active
  • Popularity: 38 (340 hits per day)
CentOS as a group is a community of open source contributors and users which started in 2003 and has been sponsored by Red Hat since 2014. CentOS Linux versions up to CentOS Linux 8 are 100% compatible rebuilds of Red Hat Enterprise Linux, in full compliance with Red Hat's redistribution requirements. In 2020 it was announced CentOS Linux is being discontinued and replaced with CentOS Stream, a developer-focused distribution which acts as a middle-stream between Fedora and Red Hat Enterprise Linux.

 Have to split ratings up but for server it is a good minimal option.
For desktop the experience is awful. Failed to work with audio drivers and while I'm technical enough I prefer not to have to rerun and modify the operating system files too much (If at all) as a desktop user.

The OS also splits itself up into two partitions on install and the install process on at least KDE was sub optimal. I found this more annoying for a smooth setup.

That is why I'm giving it a low rating of 3. Recommend another distro for desktop. That said as a server for app hosting it could work just fine.
